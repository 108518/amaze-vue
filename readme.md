## Amaze Vue

基于Amaze UI 和 Vue2 封装的组件库


### 未经过严格测试……请勿在生产环境中使用

### 开发
```bash
    npm run dev
```


### build
```bash
    npm run build
```

### 使用

```bash
    npm install amaze-vue --save
```

```js
	import Vue from 'vue';
	import Amazeui from 'amaze-vue';
	import 'amaze-vue/dist/amze-vue.css';

	Vue.use(Amazeui);
```
