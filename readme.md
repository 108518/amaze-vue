### [BUG 反馈](https://github.com/sunshineJi/amaze-vue/issues)


### 开发
```bash
    npm run dev
```


### build
```bash
    npm run build
```

### 使用

```bash
    npm install amaze-vue --save
```

```js
	import Vue from 'vue';
	import Amazeui from 'amaze-vue';
	import 'amaze-vue/dist/amaze-vue.css';

	Vue.use(Amazeui);
```
