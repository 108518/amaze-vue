## Amaze Vue

基于Amaze UI 和 Vue2 封装的组件库


### 文档(补充中……) <https://sunshineji.github.io/amaze-vue-docs/>


### 开发
```bash
    npm run dev
```


### build
```bash
    npm run build
```

### 使用

```bash
    npm install amaze-vue --save
```

```js
	import Vue from 'vue';
	import Amazeui from 'amaze-vue';
	import 'amaze-vue/dist/amaze-vue.css';

	Vue.use(Amazeui);
```
